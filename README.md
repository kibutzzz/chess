# Chess Game

**Work in Progress**

A terminal-based chess game implementation written in Java.

## Overview

This project implements a chess game engine with object-oriented design principles. The game features a complete chess board representation with all standard chess pieces and their movement rules.

## Technical Details

- **Language**: Java 21
- **Build Tool**: Maven
- **Testing**: JUnit 5 with Mockito
- **Code Style**: Google Java Format with Spotless plugin
- **Dependencies**: Lombok for reducing boilerplate code

## Building and Running

### Prerequisites
- Java 21 or higher
- Maven 3.6+

### Build
```bash
mvn clean compile
```

### Run Tests
```bash
mvn test
```

### Code Formatting
```bash
mvn spotless:apply
```

## Development

This project uses:
- **Lombok** annotations to reduce boilerplate code
- **Spotless** for consistent code formatting
- **JUnit 5** and **Mockito** for comprehensive testing
